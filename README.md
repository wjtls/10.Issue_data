# 10.Issue_data

한국, 해외이슈 지표화 및 메신저 전송
<br>
<br>
<br>
기능
1. 해외, 한국 뉴스 API 활용 (해외:newsapi, 국내:네이버금융)
2. 번역 기능
3. 긍정 키워드, 부정 키워드별 이슈 데이터 수집
4. 불러올 기간 설정
5. 뉴스강도 , 정량화하여 지표로 활용가능
